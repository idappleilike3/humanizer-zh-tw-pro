# Humanizer zh-tw Pro

> 把 AI 生成的文字改寫成「台灣繁中人」自然語氣，消除 AI 味與翻譯腔。

---

## 為什麼需要這個 Skill？

AI 寫的中文常常有這些毛病：

- **AI 味詞彙**：「讓我們一起」「值得注意的是」「不容錯過」「開啟...新篇章」
- **翻譯腔句型**：長複合句、被動句過多、「對...進行...」
- **簡中 → 繁中** 的隱形翻譯：「軟件」「視頻」「數據」「雲端」→「軟體、影片、資料、雲端」是台灣人講的
- **沒人味**：不像人講話、不像 LINE 訊息

這個 Skill 把上面這些毛病一次掃掉。

## 處理流程（4 步）

```
Step 1 識別 AI 味痕跡 → Step 2 用潤稿 prompt 改寫 → Step 3 自動驗證 → Step 4 雙鏡頭檢驗
```

### Step 1 — 識別 AI 味

常見 AI 痕跡：
- 句式：「讓我們一起...」「值得注意的是...」「不容錯過的...」
- 結構：「首先...其次...最後...」「總的來說...」「綜上所述...」
- 翻譯腔：「被動句過多」「對...進行...」
- 歐化句：「被...所...」

### Step 2 — 套用潤稿 prompt（給底層 LLM）

```
你是台灣繁中人口語教練。把下列文字改寫成台灣人自然說話的語氣：

要求：
1. 短句優先，不要長複合句
2. 避免翻譯腔與 AI 味詞彙
3. 保留原意，不誇張、不誇飾
4. 適度口語（可加「啊」「喔」「耶」「嘛」「欸」）
5. 標點統一中文標點
6. 維持品牌語氣
```

### Step 3 — 自動化驗證

- 禁用詞掃描（10+ 客戶面向禁詞 + 10+ AI 味詞彙）
- 簡轉繁檢查（避免「軟件」「視頻」「數據」混入）
- AI 味標記掃描（輸出後再掃一次）

### Step 4 — 雙鏡頭檢驗

- 👤 **客戶端鏡頭**：讀者 3 秒能懂嗎？感受如何？
- 💼 **商業顧問鏡頭**：變現 / 規模化 / 差異化 / 風險 / 數據一致嗎？

## 適用場景

- ✅ **短劇文案潤稿**（房仲 / 醫美 / 美容產業）
- ✅ **LINE 官方帳號 / 客戶回覆**：口語化、自然、不官腔
- ✅ **報價單 / SOP 文件**：人味化、不冰冷
- ✅ **行銷文案 / 銷售頁**：去掉 AI 味、建立信任感
- ✅ **Threads / Instagram 貼文**：自然社群風、避免小編味
- ✅ **Landing Page CTA**：明確行動詞、具象稀缺

不適用：
- ❌ 學術論文 / 正式公文（保留學術腔體）
- ❌ 程式碼註解（用 `code-humanizer` 之類專用 skill）
- ❌ AI 英文內容（這是台灣繁中專用）

## 設計原則

### 1. 風格基於真實禁用詞清單
